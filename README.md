### GUNMAN

![スクリーンショット 2023-09-18 145824](https://github.com/yu03040/EngineerPortfolio/assets/131416689/0617dcf8-80a7-48a1-b2f6-2b2928bdc3c8)

- 今まで、マウス操作でしか UI を操作できないゲームしか作らなかったので、ゲームパッドで動かせるようにし操作性を改善しました。
- C++ と Blueprint の使い分けをしました。具体的には AI 周りは Blueprint で、それ以外（Blueprint で使いたい関数や各パラメータの初期値の設定など）は C++ で開発しました。
- キャラクターの走る速さを、滑らかになるように Curve Float と C++ で調整しました。これで、走っているときは段々速く、走っていないときは段々通常の速度になるようにしました。

## 操作説明

### Endless Run

- ジャンプ：スペースバー
- 移動：A, D, 十字キー左, 十字キー右
- ポーズ；Q
- 道を曲がるとき（左右）：A, D, 十字キー左, 十字キー右

### First Person

- 移動 : W, A, S, D 、ゲームパッド左スティック
- ジャンプ : スペースバー、ゲームパッド B ボタン
- アイテムを拾う : E 、ゲームパッド A ボタン
- イベントリを開く : I 、ゲームパッド Y ボタン
- FPS と TPS の切り替え : P 、ゲームパッド左ショルダー
- キャラクターの回転：マウスの移動、ゲームパッド右スティック
- 銃を撃つ：マウスの左ボタン、ゲームパッド右トリガー

### Big Or Small

- ゲーム開始 : Enter キー
- Small(0-4) : 1
- Big(5-9) : 2
- 再挑戦[はい] : 1
- 再挑戦[いいえ] : 2

### Battery Collector

- 移動 : W, A, S, D 、ゲームパッド左スティック
- 視点移動 : マウス、ゲームパッド右スティック
- ジャンプ : スペースバー、ゲームパッド A ボタン
- バッテリーの回収 : C 、ゲームパッド B ボタン
